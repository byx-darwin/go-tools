**Goal:** 创建单一综合 example 项目，验证 go-tools 四层库全部功能，包含 Hertz HTTP + Kitex RPC + 自动化测试报告

**Architecture:** 单一 Go module 位于 `example/`，通过 `replace` 引用本地四层库。Hertz HTTP server 暴露 30+ 路由演示各包功能，Kitex RPC server 演示日志/可观测性集成。测试运行器为独立进程，启动 server 后逐端点验证。

**Tech Stack:** Go 1.25+, Hertz, Kitex, Protobuf, kafka-go, go-redis v9, go-elasticsearch v8, clickhouse-go v2, oops, samber/hot, polaris-go

## Global Constraints

- Go 1.25+，workspace 模式（go.work）
- 模块边界：go-common 无框架依赖，go-middleware 无 Hertz/Kitex 依赖，go-framework 依赖前两者
- 错误处理使用 `oops` 风格，错误码：go-framework 10000-10499，go-middleware 20000-20699，go-auth 40000-40099
- 导出符号必须有 godoc 注释
- 使用 `any` 替代 `interface{}`
- 时间配置使用 `time.Duration`，YAML 支持 `30s` 格式
- 所有新代码必须通过 `gofmt`、`go vet`、`golangci-lint`

```go
// AppConfig 应用全局配置。
type AppConfig struct {
	Server        ServerConfig                `yaml:"server"`
	Log           config.Config               `yaml:"log"`
	JWT           JWTConfig                   `yaml:"jwt"`
	StoreMode     string                      `yaml:"store_mode"`
	Redis         redis.Config                `yaml:"redis"`
	Kafka         kafka.WriterConfig          `yaml:"kafka"`
	KafkaReader   kafka.ReaderConfig          `yaml:"kafka_reader"`
	DB            db.Config                   `yaml:"db"`
	Elasticsearch es.Config                   `yaml:"elasticsearch"`
	ClickHouse    clickhouse.Config           `yaml:"clickhouse"`
	Captcha       config.CaptchaOption        `yaml:"captcha"`
	Hertz         hertzConfig.ServerConfig    `yaml:"hertz"`
	Kitex         kitexConfig.ServerConfig    `yaml:"kitex"`
	Observability config.ObservabilityConfig  `yaml:"observability"`
	Polaris       PolarisConfig               `yaml:"polaris"`
}

// ServerConfig 服务地址配置。
type ServerConfig struct {
	HTTPAddr string `yaml:"http_addr"`
	RPCAddr  string `yaml:"rpc_addr"`
}

// JWTConfig JWT 配置。
type JWTConfig struct {
	Secret            string          `yaml:"secret"`
	Issuer            string          `yaml:"issuer"`
	AccessExpiration  config.Duration `yaml:"access_expiration"`
	RefreshExpiration config.Duration `yaml:"refresh_expiration"`
}

// PolarisConfig 北极星配置中心。
type PolarisConfig struct {
	Enabled   bool   `yaml:"enabled"`
	Namespace string `yaml:"namespace"`
	FileGroup string `yaml:"file_group"`
	FileName  string `yaml:"file_name"`
}

// LoadConfig 加载配置并展开环境变量。
func LoadConfig(path string) (*AppConfig, error) {
	data, err := os.ReadFile(path)
	if err != nil {
		return nil, fmt.Errorf("read config: %w", err)
	}
	// 展开环境变量（如 ${OTEL_ENDPOINT}）
	expanded := os.ExpandEnv(string(data))
	var cfg AppConfig
	if err := yaml.Unmarshal([]byte(expanded), &cfg); err != nil {
		return nil, fmt.Errorf("parse config: %w", err)
	}
	return &cfg, nil
}
```

```go
package main

import (
	"context"
	"fmt"
	"os"
	"os/signal"
	"syscall"

	"github.com/byx-darwin/go-tools/go-common/log"
	"github.com/byx-darwin/go-tools/go-framework/hertz"
	hertzlog "github.com/byx-darwin/go-tools/go-framework/hertz/log"
	hertzobs "github.com/byx-darwin/go-tools/go-framework/hertz/observability"
	kitexlog "github.com/byx-darwin/go-tools/go-framework/kitex/log"
	kitexobs "github.com/byx-darwin/go-tools/go-framework/kitex/observability"

	"github.com/cloudwego/hertz/pkg/app/server"
	klog "github.com/cloudwego/kitex/pkg/klog"
	"github.com/cloudwego/kitex/pkg/rpcinfo"
	kserver "github.com/cloudwego/kitex/server"
)

func main() {
	ctx, cancel := context.WithCancel(context.Background())
	defer cancel()

	// 1. 加载配置
	cfg, err := LoadConfig("config.yaml")
	if err != nil {
		fmt.Fprintf(os.Stderr, "load config: %v\n", err)
		os.Exit(1)
	}

	// 2. 初始化日志
	if err := log.Init(cfg.Log, log.ReleaseInfo{
		ServiceName: "go-tools-example",
		Version:     "v1.0.0",
		Environment: "development",
	}); err != nil {
		fmt.Fprintf(os.Stderr, "init log: %v\n", err)
		os.Exit(1)
	}
	defer log.Close()

	// 替换 Hertz 和 Kitex 默认日志
	hlogAdapter := hertzlog.NewHertzAdapter(log.L())
	server.WithHlog(hlogAdapter) // 注：需确认 API
	klog.SetLogger(kitexlog.NewKitexAdapter(log.L()))

	// 3. 初始化可观测性
	otelProvider, err := hertzobs.NewProvider(ctx, cfg.Observability)
	if err != nil {
		log.L().Warn("observability init failed, continuing without tracing", "error", err)
	}
	defer func() {
		if otelProvider != nil {
			_ = otelProvider.Shutdown()
		}
	}()

	kitexProvider, err := kitexobs.NewProvider(ctx, cfg.Observability)
	if err != nil {
		log.L().Warn("kitex observability init failed", "error", err)
	}
	defer func() {
		if kitexProvider != nil {
			_ = kitexProvider.Shutdown()
		}
	}()

	// 4. 初始化依赖（session store, device store, middleware clients）
	deps := initDeps(cfg)

	// 5. 创建 Hertz server
	h := createHertzServer(cfg, deps, otelProvider)

	// 6. 创建 Kitex server（goroutine 中运行）
	go startKitexServer(ctx, cfg, deps, kitexProvider)

	// 7. 启动 Hertz
	go h.Spin()

	// 8. 优雅关闭
	quit := make(chan os.Signal, 1)
	signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
	<-quit
	log.L().Info("shutting down...")
	cancel()
	h.Shutdown(context.Background())
	kserver.Shutdown()
}
```

```go
package handler

import (
	"github.com/cloudwego/hertz/pkg/app"
	"github.com/cloudwego/hertz/pkg/app/server"
	"github.com/cloudwego/hertz/pkg/protocol/consts"
	"github.com/byx-darwin/go-tools/go-common/crypto"
	hertzresp "github.com/byx-darwin/go-tools/go-framework/hertz"
)

// RegisterCryptoRoutes 注册 crypto 演示路由。
func RegisterCryptoRoutes(h *server.Hertz) {
	h.GET("/common/crypto", handleCrypto)
}

func handleCrypto(ctx context.Context, c *app.RequestContext) {
	data := []byte("hello go-tools")
	teaKey := "0123456789abcdef"

	result := map[string]string{
		"md5":       crypto.MD5(data),
		"sha256":    crypto.SHA256(data),
		"sha512":    crypto.SHA512(data),
		"hmac256":   crypto.HMACSHA256(data, []byte("secret")),
	}

	// TEA 加解密
	encrypted, pad, err := crypto.EncodeTeaStr(data, teaKey)
	if err == nil {
		decrypted, decErr := crypto.DecodeTeaStr(encrypted, pad, teaKey)
		if decErr == nil {
			result["tea_encrypted"] = string(encrypted)
			result["tea_decrypted"] = string(decrypted)
		}
	}

	hertzresp.Success(c, result)
}
```

```go
package middleware

import (
	"github.com/byx-darwin/go-tools/example/handler"
	"github.com/byx-darwin/go-tools/go-auth/device"
	"github.com/byx-darwin/go-tools/go-auth/session"
	"github.com/byx-darwin/go-tools/go-framework/hertz/middleware"
	"github.com/cloudwego/hertz/pkg/app/server"
)

// RegisterMiddleware 注册全局中间件。
func RegisterMiddleware(h *server.Hertz, sessionStore session.Store, deviceStore device.Store, jwtSecret []byte) {
	h.Use(
		middleware.AccessLog(),
		middleware.Cors(),
	)
	_ = sessionStore
	_ = deviceStore
	_ = jwtSecret
}

// RegisterProtectedRoutes 注册受保护路由。
func RegisterProtectedRoutes(h *server.Hertz, sessionStore session.Store, deviceStore device.Store, jwtSecret []byte) {
	// /protected/jwt — JWT 中间件保护
	jwtGroup := h.Group("/protected", middleware.JWTAuth[handler.AppClaims](jwtSecret))
	jwtGroup.GET("/jwt", handler.HandleProtectedJWT)

	// /protected/session — Session 中间件保护
	sessGroup := h.Group("/protected", middleware.SessionAuth(sessionStore))
	sessGroup.GET("/session", handler.HandleProtectedSession)

	// /protected/device — Device Auth 中间件保护
	devGroup := h.Group("/protected",
		middleware.JWTAuth[handler.AppClaims](jwtSecret),
		middleware.DeviceAuth(deviceStore, handler.ExtractDeviceClaims),
	)
	devGroup.GET("/device", handler.HandleProtectedDevice)
}
```
